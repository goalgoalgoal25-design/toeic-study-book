# TOEIC スタディブック

レベル別の目標点数・課題・問題演習・AI問題生成(Gemini連携)が一体になった、自分専用のTOEIC学習アプリです。

## 🌐 どこからでも使える公開版(スマホOK)

**https://goalgoalgoal25-design.github.io/toeic-study-book/**

スマホでもPCでも、外出先のモバイル回線でも上のURLで開けます。スマホでは「ホーム画面に追加」しておくと便利です。
※ 学習記録とAPIキーは端末(ブラウザ)ごとに保存されます。

## 開き方(かんたん)

**「アプリを起動.bat」をダブルクリック**してください。黒い画面(サーバー)が開き、ブラウザでアプリが立ち上がります。黒い画面を閉じるとアプリも止まります。

※ うまくいかない場合は `index.html` をダブルクリックしても開けます(ほぼすべての機能が動きます)。

## 使い方の流れ

1. **🩺 診断テスト** … 12問で現在地(推定スコア)と始めるレベルを判定
2. **🏠 ホーム** … レベル別の目標点数マップ(400 / 500 / 600 / 730 / 860 / 900+点)
3. **✅ 学習課題** … 選んだレベルの週次課題をチェックしながら進める
4. **📝 問題演習** … 内蔵のレベル別問題(Part 2 音声 / Part 5 / Part 7)
5. **🤖 AI問題生成** … Gemini があなたの弱点に合わせて新作問題を生成(下記)
6. **📊 記録** … 正答率・間違えた問題・レベルアップ判定・再診断リマインド

各レベルで**累計30問以上・正答率80%以上**になると「レベルアップ判定」が出ます。

## AI問題生成(Gemini)の設定

2通りの使い方があります。

### A. APIキー方式(全自動・Part 1 写真生成が使える)

1. [Google AI Studio](https://aistudio.google.com/apikey) にGoogleアカウントでログイン
2. 「Create API key」でキーを発行(無料枠あり)
3. アプリの「🤖 AI問題生成」タブでキーを貼り付けて保存

これで Part 1(写真をAIが生成+音声読み上げ)/ Part 2 / Part 5 の新作問題がワンクリックで作れます。

### B. コピペ方式(Gemini Pro 加入をそのまま使う・APIキー不要)

「🤖 AI問題生成」タブ下部の「コピペ連携」から、プロンプトをコピーして
[gemini.google.com](https://gemini.google.com) に貼り付け、返ってきたJSONをアプリに貼り戻すと演習が始まります(Part 2 / Part 5 対応)。

## リスニング音声について

問題演習・AI問題生成の画面上部にある「🔊 リスニング音声」で切り替えられます。

- **Gemini音声(おすすめ・要APIキー)** … ネイティブ品質。本番同様に **🇺🇸米・🇬🇧英・🇦🇺豪・🇨🇦加の4アクセント**と**男性/女性の話者**を切り替え(またはランダム)できます。
- **ブラウザ内蔵** … APIキーなしで動く簡易音声。英語ボイスを自動選択します(英語ボイスが見つからない場合はメッセージが出ます)。

再生後に「話者: 🇬🇧 イギリス・女性」のように、いま聞いた音声の国・性別が表示されます。

## メモ

- 学習データはこのパソコンのブラウザ内(localStorage)にのみ保存されます。
- 問題はすべてオリジナルで、実際のTOEIC過去問ではありません。
- レベル基準は IIBC の CEFR対照表 / PROFICIENCY SCALE を参考にしています。
